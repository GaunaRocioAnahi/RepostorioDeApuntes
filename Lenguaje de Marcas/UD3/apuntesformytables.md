----------------------------------------------
listas anidada:

```html
<ol>
  <li>Elemento uno</li>
  <li>Elemento dos
    <ol type="a">
      <li>Subelemento dos-a</li>
      <li>Subelemento dos-b</li>
    </ol>
  </li>
  <li>Elemento tres</li>
</ol>
```
----------------------------------------------
listas desordenada:


```html
<ul style="list-style-type: square;">
  <li>Elemento uno</li>
  <li>Elemento dos</li>
  <li>Elemento tres</li>
</ul>
```
atributos para `style="list-style-type: disc` --> valor por defecto
atributos para `style="list-style-type: circle` --> Circulo hueco
atributos para `style="list-style-type: square`  --> Cuadrado.
